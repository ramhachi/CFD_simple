# FSAE 簡易流体解析システム (OpenFOAM + Docker)

## 概要
このシステムは、FSAEの空力パッケージや車体形状の解析を、複雑な設定なしに直感的に実行するための「対話型CFDツール」です。

## 必須環境
- Docker Desktop (Mac環境で起動していること)

## 使い方

1. **STLファイルの準備**
   - 新規作成される `models/` フォルダの中に、解析したいSTLファイルをすべて置きます。
   - **空力部品**: `body.stl` などの任意の名前。（複数可）
   - **タイヤ**: `tires.stl` という名前で保存してください（対話機能で処理を分けるため）。
   - **フレーム類**: `frame.stl` という名前で保存してください（同様）。

2. **解析の実行**
   Pythonなどのローカルでの面倒な環境構築は不要です。すべてDockerコンテナ内で動作します。
   お使いのOSに応じた以下のスクリプトをダブルクリック、またはターミナルから実行してください。

   **Windowsの場合:**
   ```bat
   run_windows.bat
   ```

   **Mac / Linux の場合:**
   ```bash
   ./run_mac.sh
   ```

3. **入力パラメータの設定**
   画面の指示に従い、順次質問に答えます。
   - 風速 (m/s)
   - メッシュの粗さ (1:粗い 2:普通 3:細かい)
   - 各STLパーツの位置オフセット（X, Y, Z方向への移動量[mm]）
   - 全体のグラウンドクリアランス（Z方向の追加移動量[mm]）

## 可視化と動画生成 (Visualization)

### A. ParaViewを使った独自の可視化
計算が成功すると、`CFD_simple/VTK/` フォルダの中にVTK形式のデータが保存されます。これを **ParaView** というフリーソフトで読み込むことで、3Dの流体結果を詳細に可視化できます。

1. **ParaViewを起動** し、左上の `File > Open` をクリックします。
2. 出力された `CFD_simple/VTK/` フォルダ内の、見たいステップ（例: `data_300/`）のファイルを選びます。
   - **空間全域の断面を見たい場合:** `data_300/internal.vtu`
   - **車体表面の圧力だけ見たい場合:** `data_300/boundary/body.vtp`
3. 左側のPropertiesパネルにある **「Apply」ボタン** を緑色から灰色になるまでクリックします。
4. 画面上部のツールバーで「Solid Color」となっているドロップダウンを **`p` (圧力)** または **`U` (速度)** に変更します。
5. **断面の切り方（Slice）**:
   - 画面上部にある「赤い四角形に白い点線」のアイコン（Slice）をクリックします。
   - Propertiesパネルで `Z Normal` （地面と平行）や `Y Normal` （車両中心断面）を選び、**「Apply」** を押すと任意の場所でスパッと切った断面が表示されます。

### B. 自動動画・断面生成機能 (`make_video.py`)
Pythonを用いて、イテレーション（計算ステップ）ごとの変化を動画 (GIF) にしたり、計算結果の静止画や、断面モデルをスイープ(スキャン)していく様子を自動抽出します。

**（※現在、このプロセスはバッチファイルによる解析完了時に自動的に実行されるため、ユーザーが個別に実行する必要はありません。結果は `visualization_output/` フォルダに保存されます）**
